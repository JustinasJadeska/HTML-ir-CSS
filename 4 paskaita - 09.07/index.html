CSS stilių prioritetai nuo svarbiausio iki mažiau svarbaus

  1. !important - retai naudojamas, geriausia nenaudoti
  2. Inline - 1000
  3. ID - 100
  4. Class - 10
  5. Element - 1
  6. CSS order (Svarbu jeigu vienoda specifikacija)
  7. User Agent Stylesheets
  8. Inheriting

Esant dviem selektoriam, kurie nusako to paties elemento vienodus stilius, yra atsižvelgiama į tų selektorių specifikacijas. Aukštesnį balą turintis selektorius turi aukštesnį prioritetą.

Esant dviem vienodos specifikacijos selektoriams keičiant tą patį stilių, yra atsižvelgiama į SCC eiliškumą. Kuris selektorius yra aprašytas vėliau, tas turės aukštesnį prioritetą.

Jeigu įmanoma nenaudoti!!! '!important' prierašo. !important prierašas ignoruoja visus prioritetus (neveikia su paveldėjimu) ir suteikia nurodytą stilių.
