CSS parašymo būdai

  • inline - rašomas elemento viduje
  • internal - rašomas HTML'e
  • external - rašoma atskirame faile, kuris turi būti prijungtas

Inline

Tag'o atidaromoje dalyje naudojamas atributas "style", kuriame priskiriamia linijiniai stiliai.

Pavyzdys

Internal

Head tagp'o viduje rašomas style tag'as, kurio viduje galima parašyti CSS stilius su tvarkinga/pilna CSS sintakse: selektorius + ribos + stiliai

Pavyzdys

External

CSS Selektoriai

  • Element - nurodomas tag'o pavadinimas
  • CLASS - klasė sukuriama parašant tašką ir pavadinimą be tarpų.
  • ID - HTML ID gali būti priskirtas elementui, kaip atributas. CSS ID sukuriamas parašant grotelių simbolį(#). Pvadinimas turi būti vienas žodis.

Pavadinimų kūrimas

  • camelCase - sekantys žodžiai rašomi iš didžiosios raidės.
  • snake_case - sekantys žodžiai rašomi už apatinio brūkšnelio.
  • kebab-case - sekantys žodžiai rašomi už minuso ženklo.
