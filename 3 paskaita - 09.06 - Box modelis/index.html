Semantics and page layouts

Kuriant internetinę svetainę arba aplikaciją, reikėtų naudoti semantinius tag'us, tam, kad būtų tvarkinga puslapio struktūra ir joje programuojant būtų lengviau susigaudyti.

  • Header
  • Nav
  • Main
  • Section
  • Article
  • Figure - skirats talpinti image, video, audio, map, gif, map iframe
  • Aside - šoninė juosta (dažnu atveju naudojamas papildomam menu).
  • Footer

Box model

Kiekvienas puslapio lelemntas turi savo 'box' modelį, kuris susideda iš žemiau išvardintų dalių.

  1. Content - Turinys
  2. Padding - Pamušalas
  3. Border - Rėmelis
  4. Margin - Paraštės (atstumai iki kitų elementų)
  5. Position - Pozicijų atitraukimai
Box model example

Padding

Padding nustatymas nurodo per kokį atstumą turinys turi būti atitrauktas nuo rėmlio.

  • padding: 5px; visiems
  • padding-top: 5px;
  • padding-right: 5px;
  • padding-bottom: 5px;
  • padding-left: 5px;
  • padding: 5px 10 px; top and bottom, right and left.
  • padding: 20px 10px 15px;
  • padding: 20px 10px 15px 20px; top, right, left, bottom.

Margin

Margin nustatymas nurodo per kokį atstumą content+padding+border turi būti atitraukti nuo aplinkinių elementų.

  • margin: 0 auto; centruoja block'ą, būtinai turi būti nustatytas, kad tai yra blokinis elementas, linijinamie neveikia (inline block).
  • margin: 5px; visiems
  • margin-top: 5px;
  • margin-right: 5px;
  • margin-bottom: 5px;
  • margin-left: 5px;
  • margin: 5px 10 px; top and bottom, right and left.
  • margin: 20px 10px 15px;
  • margin: 20px 10px 15px 20px; top, right, left, bottom.

Border

Rėmelis (border) yra kažkokio tipo, storio ir spalvos juosta skirianti margin ir padding.

  • border: line-width, line type, line-color
  • border-top-width: 10px;
  • border-width: 10px;
  • border-style: none / hidden / solid / double / dotted / dashed / inset / outsed / groove / ridge;
  • border-right-style: ;
  • border-color: red;
  • border-bottom-color: black;
  • border-radius: px / %;
  • border-top-right-radius: px / %;

Selectors

  • Element/Tag
  • CLass
  • ID
  • * - viskas
  • element1 element2 - elementas2 esantis elemento1 viduje, nesvarbu kaip toli
  • element1 > element2 - elementas 2 esanti vaikinis elementas elemento1
  • element.className - elementas turintis klasę pavadinimu className
  • element#idName - elementas turintis ID pavadinimu idName
  • selector, selector - keletas selektorių su vienodais stiliais
  • element1 + element2 - tame pačiame lygmenyje iš karto už element1 einantis element2
  • element1 ~ element2 - tame pačiame lygmenyje už element1 einantys visi element2 nesvarbu kaip toli
  • selector:hover - pelė užėjusi ant selektinio elemento.
