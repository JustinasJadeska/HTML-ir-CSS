Atributai

  1. rel - relationship
  2. href - hyper link reference
  3. src - source
  4. alt -alternative text
  5. target - atidaro nuorodą naujame lange
  6. class - skirtas priskirti elementui klasę
  7. id - skiras priskirti elementui id
  8. style - nurodo elemento stilių linijiniu būdu

Klaviatūros trumpinių sąrašas

  1. CTRL X - iškerpa
  2. CTRL Z - atšaukia paskutinį veiksmą
  3. CTRL Y - grąžina paskutinį veiksmą
  4. CTRL A - viską pažymi
  5. CTRL S - išsaugoti failą
  6. CTRL L - pažymi eilutę, jei daugiau spaudi pažymi daugiau
  7. CTRL SHIFT K - ištrinti eilutę, kurioje yra 'cursor'
  8. SHIFT ALT - pakeičia kalbą
  9. ALT TAB - padeda greitai pakeisti atidarytus langus kompiuteryje
  10. CTRL F - atidaro paieškos lauką
  11. CTRL G - nurodyti į kurią kodo eilutę nori būti nukeltas
  12. SHIFT ALT UP/DOWN - dublikuoja eilutę aukštyn/žemyn
  13. CTRL ENTER - sukuria naują eilutę apačioje
  14. ALT UP/DOWN - nukelia eilutę aukštyn/žemyn
  15. CTRL D - parenka visus dublikatus, to kas yra pažymėta
  16. WINDOW TAB - galimybė sukurti papildomus langus
  17. CTRL / - rašyti komentarus
