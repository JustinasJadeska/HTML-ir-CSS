Grid

  • Parent (container)
    • display:grid/inline-grid
    • grid-template-columns/rows: unit/s - nurodai kiek ir kokio dydžio stulpelių/eilučių nori savo kuriamame tinklelyje
    • grid-template: grid-template-rows / grid-template-columns
    • grid-template-areas: "name|nameOtherOrSame|more|..." "kitaEilute|.|none|..." -
    • column-gap/row-gap - nurodo tarpus tarp tinklelio stulpelių arba eilučių
    • gap: row-gap column-gap/
    • justify-items: eina antras
    • align-items: eina pirmas
    • place-items: align-items justify-items - junginys
    • justify-content:...; X ašyje
    • align-content:...; nusako kaip visas tinklelis išsidėsto konteinerio viduje Y ašyje
    • place-content: align-content justify-content
    • grid-auto-rows:
    • grid-auto-columns:
    • grid-auto-flow: row/column/dense/row dense/column dense - nusako kaip elementai išsidėsto jeigu jiems nėra priskirtos vietos
    • grid: grid-template-rows / grid-template-columns / grid-template-areas / grid-auto-rows / grid-auto-columns / grid-auto-flow;
  • Child (elements in container)
    • grid-column-start: number;
    • grid-column-end: number/spanNumber;
    • grid-row-start: number
    • grid-row-end: number/spanNumber;
    • grid-row: grid-row-start grid-row-end;
    • grid-column: grid-column-start grid-column-end
    • grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end
    • justify-self:
    • align-self:
    • place-self: align-self justify-self;
  • Units and Functions
    • fr - frakcija, nurodo kokią dalį pločio/aukščio turi priskirti
    • repeat(kartai, kiekis) - nurodo kiek kartų kartoti tam tikrą kiekio nurodymą
    • minmax(min kiekį, max kiekį) - nurodo kiek vietos tinklelio (grid) eilutė ar stulpelis gali užimti
    • fit-content()
    • min()
    • max()
    • masonry
    • subgrid
